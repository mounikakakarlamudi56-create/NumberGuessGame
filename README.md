# Number Guessing Game

## Overview
This is a console-based Number Guessing Game developed in Java. The program generates a random number between 1 and 100, and the player attempts to guess it within a limited number of attempts. The game provides hints after each guess and includes multiple difficulty levels, score calculation, a timer, input validation, and a replay option.

## Features
- Random number generation (1–100)
- Easy, Medium, and Hard difficulty levels
- Too High / Too Low hints
- Attempt counter
- Score calculation
- Time tracking
- Input validation
- Play Again option
- Clean and user-friendly console interface
